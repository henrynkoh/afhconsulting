# AFH Success Consultant Curriculum

A comprehensive Next.js application designed to train AFH Success Consultants in operating financially sound and long-term successful Adult Family Homes.

## Overview

This curriculum provides a complete "cookbook" for launching and maintaining a high-profit, private-pay-heavy Adult Family Home. The platform includes:

- **6 Core Modules** covering business foundation, site selection, pricing, marketing, billing, and compliance
- **Interactive Tools** including market analysis templates, checklists, and calculators
- **Marketing Resources** with scripts, menus, and activity schedules
- **Certification Exam** to validate consultant expertise
- **Comprehensive Handbook** consolidating all curriculum content

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm start
```

## Key Concepts

### The 5-Mile Rule
Private-pay families typically choose homes within 5 miles of their residence or office.

### Daily Rate Calculation
```
Daily Rate = (Monthly Rate × 12) ÷ 365
```

### Financial Health Metrics
- Operating Reserve: 6 months expenses
- Private Pay Ratio: 80%+ of residents
- Occupancy Rate: 95%+
- Net Profit Margin: 25-35%
