# AFH Success Consultant Curriculum - Quick Start Guide

## Get Started in 5 Minutes

This quick start guide will have you up and running with the AFH Success Consultant Curriculum in minutes.

## Prerequisites Check (1 minute)

Verify you have Node.js installed:
```bash
node --version
# Should show v18.0.0 or higher
```

If not installed: [Download Node.js](https://nodejs.org/)

## Installation (2 minutes)

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open Browser
Navigate to: **http://localhost:3000**

## First Steps (2 minutes)

### 1. Explore the Homepage
- Review the 6 module cards
- Check out additional resources
- Familiarize yourself with navigation

### 2. Start Module 1
- Click on "Module 1: Business Foundation"
- Read the module goal
- Begin with "Financial Reality Check"

### 3. Use Navigation
- Use "Next Module" button to progress
- Use "Back to Home" to return
- Bookmark important sections

## Essential Commands

```bash
# Development
npm run dev          # Start dev server (http://localhost:3000)

# Production
npm run build        # Build for production
npm start            # Start production server

# Maintenance
npm run lint         # Check for errors
```

## Troubleshooting

### Port Already in Use
```bash
# Use different port
PORT=3001 npm run dev
```

### Module Not Found
```bash
# Reinstall dependencies
rm -rf node_modules
npm install
```

### Styling Issues
```bash
# Clear cache and restart
rm -rf .next
npm run dev
```
